# 💾 Cyber-Floppy Quest (BASIC)

[PL] Klasyczna, tekstowa gra strategiczno-klikana napisana w retro języku BASIC (kompatybilna z QBasic / GW-BASIC). Zarządzaj energią, zbieraj dane i ulepszaj swój cyber-sprzęt!

[EN] A classic text-based strategy/clicker game written in retro BASIC (compatible with QBasic / GW-BASIC). Manage your energy, gather data, and upgrade your cyber-gear!

---

## 🇵🇱 

### Funkcje gry
* **Prawdziwe Retro:** Kod napisany zgodnie ze starą szkołą, z użyciem numeracji linii (`10`, `20`, `30`...).
* **Gospodarka Zasobami:** Musisz balansować między hakowaniem (generuje punkty, zabiera energię) a odpoczynkiem.
* **Skalowanie Poziomu:** Każde ulepszenie pamięci RAM zwiększa zysk z hakowania, ale kosztuje coraz więcej danych.
* **Kolorowy interfejs tekstowy:** Wykorzystanie instrukcji `COLOR` dla nadania cyberpunkowego klimatu w terminalu.

### Jak to uruchomić?
1. Pobierz emulator retro, np. **PC-BASIC** lub **DOSBox** z uruchomionym `QBASIC.EXE`.
2. Skopiuj powyższy kod i zapisz jako plik `CYBER.BAS`.
3. Otwórz plik w środowisku BASIC i wciśnij klawisz `F5`, aby uruchomić grę!

---

## 🇬🇧 

### Game Features
* **True Retro Experience:** Old-school code style utilizing classic line numbering (`10`, `20`, `30`...).
* **Resource Management:** Balance your choices between hacking (grants data, drains energy) and resting.
* **Progression System:** Buying RAM upgrades boosts your hacking efficiency but costs exponentially more data.
* **Colorful Text UI:** Uses the native `COLOR` statements to deliver an authentic 80s/90s cyberpunk terminal vibe.

### How to Run
1. Download a retro interpreter/emulator like **PC-BASIC** or run `QBASIC.EXE` via **DOSBox**.
2. Copy the code above and save it as `CYBER.BAS`.
3. Load the file into your BASIC environment and hit `F5` to run!
